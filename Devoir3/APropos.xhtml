À Propos

À qui s'adresse ce site

Cette page est une introduction au Parkour, le sport inventé dans notre époque. Alors ce site adresse aux gens qui ne le connais pas et aussi aux personnes qui sont déjà intéressé par cette activité et ils vont prendre des renseignements supplémentaires comme l’histoire et comment débuter.

Construction

Pour créer ce site j’ai utilisé un fichier HTML pour la page INDEX et des fichiers XHTML pour les autres pages en considérant les balisages appropriés et les regroupements nécessaires pour contrôler plus facilement le design du site par le fichier CSS. J’ai téléchargé les images gratuites pour ce travail et j’ai utilisé Wikipédia comme une source gratuite pour certains renseignements.

Critères Ergonomiques

Voici la liste des critères ergonomiques que j’ai essayé respecter :
  • Compatibilité
    • Le langage utilisé pour ce site est simple compréhensible par la majorité du monde.
    • Le site est cohérent avec les écrans de tailles différentes.
  • Guidage
    • Les contenus sont groupés et les objets différents comme les paragraphes et les titres sont distingués.
    • Chaque paragraphe est placé dans son boite guidant par son titre spécifié.
    • Il y a des liens sur les mots relié à chaque page dans la page d’accueil, distingué par de couleur différent.
    • Utilisation des couleurs énergiques reliés par le sujet du site.
    • Page sélectée est distinguée dans le menu en changement le couleur de fond.
    • Contraste entre les couleurs est respectée alors c'est bien lisible.
  • Homogénéité
    • Les fenêtres ont le même type d’agencement.
    • La syntaxe est cohérente.
  • Concision
    • Le site web ne contient que les informations pertinentes sur le sport Parkour.
    • Les paragraphes sont courts et le dialogue est simple.
