Choix de représentation

Le jeu était un bon défi pour comprendre à manipuler les structures DOM avec JQuery, de plus on s'est beaucoup amusé à coder ce jeu.
L'idée était de créer une page XHTML avec juste un formulaire, et d'ajouter et de modifier le reste grace à JavaScript et évidement JQuery.
Le plus difficile était de diviser des portions de vue de l'image entre les cellules de la table correctement.
Une fois l'image brassée, il fallait l'enlever pour qu'on fait en sorte qu'il y est un souvenir de chaque petite partie de l'image brassée. Pour le reste, le jeu se déroule tel que demandé.
En plus d'utiliser la souris, on peut utiliser les touches de direction du clavier.

Expérience

La librairie JQuery nous a été grandement utile par sa simplicité de manipulation du DOM HTML.
Cependant, nous avons rencontré certains problèmes mineures dus aux subtilités de JQuery.
Par ailleurs, jQuery nous a grandement facilité la gestion des événements.
Cependant, nous avons essayer d'utiliser pour le mieux cette librairie, pour minimiser les lignes de code en JavaScript, ce qui n'est pas
si subtile dû à sa richesse d'une part, et à sa ressemblance au code JavaScript d'une autre part.
En essayant le jeu sur les différents navigateurs, on a remarqué que le fonctionnement du jeu est légèrement différent.
Ce qu'on a constaté, c'est que FireFox et Explorer Edge permettent une meilleur exécution du jeu. On se demandait si la programmation JavaScript était limitée à ces navigateurs.
